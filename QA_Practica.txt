Команда `nano` - это очень простой редактор файла.
Синтаксис: nano пробел 'имя файла' далее Enter
Пример: nano QA_Practica

Команда 'cat' выводит на экрад содержимое файла
Синтаксис: cat пробел `имя файла` далее Enter
Пример: cat QA_Practica

pwd (print working directory) - команда выведет на экран каталог, 
где ты сейчас находишься.

ls (list directory contents) - команда выводит все дериктории

ls -l команда выводит дополнительную информацию деректорий.
Например - права доступа, дата изменения

ls -a команда выводит все файлы и папки в том числе и скрытые.

ls -t команда отсортирует список файлов по дате последнего изменения


ls -alt ТРОЙНАЯ КОМБИНАЦИЯ команда выводит скрытые файлы с подробной 
информацией и список начинается с файлов изменёных в последнюю очередь

ls -lS (верхний регистр) команда с флагом -lS сортирует файды по размеру 

cd ~ ключь (~) переведёт в домашний каталог
